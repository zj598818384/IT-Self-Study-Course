一、if-else语法
if(判断条件)
{
      执行的代码；
}
else
{
      执行的代码;
}
执行过程：程序执行到if处，首先判断if所带的小括号中的判断条件是否成立，如果成立，也就是返回一个true，则执行if所带的大括号中的代码，执行完成后，跳出if-else结构；如果if所带的小括号中的判断条件不成立，也就是返回一个false，则跳过if语句，执行else所带的大括号中的语句，执行完成后，跳出if-else结构。

if-else特点：先判断，在执行，最少都要执行一条代码。
用于两种情况呢的判断。

注意：else永远跟离他最近的if配对

二、if else-if
作用：用来处理多条件的区间性的判断
语法：
if(判断条件)
{
      执行的代码；
}
else if(判断条件)
{
      执行的代码；
}
....
else
{
      执行的代码；
}
执行过程：程序首先判断第一个if所带的小括号中的判断条件，如果条件成立，也就是返回一个true，则执行该if所带的大括号中的代码，执行完成后，立即跳出if else-if结构。
如果第一个if所带的判断条件不成立，也就是返回一个false，则继续向下进行判断，依次的判断每一个if的判断条件，如果成立，就执行该if索敌啊的大括号的代码，如果不成立，则继续向下判断，如果每个if所带的判断条件都不成立，就看当前这个if else-if结构中是否存在else。如果有else的话，则执行else中所带的代码，如果没有else，则整个if-else if什么都不做。
else可以省略
