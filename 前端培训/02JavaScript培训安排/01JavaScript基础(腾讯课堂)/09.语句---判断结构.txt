一、语句
js中的语句分为：判断结构、分支结构、循环结构。

二、判断结构
第一种：
       if(判断的条件表达式)
       {
             判断成立后执行的语句;
       }

第二种：
       if(判断的条件表达式)
       {
              判断成立后执行的语句;
       }
      else    //判断不成立之后反面剩余的所有条件都会执行else
      {
              判断不成立后执行的代码;
      }

第三种：
       if(判断的条件表达式)
       {
              判断成立后执行的语句;
       }
       else if(判断的条件表达式)
       {
              判断成立后执行的语句;
       }
